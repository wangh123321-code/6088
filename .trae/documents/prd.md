## 1. 产品概述

3D跑步姿势分析工具——帮助跑者通过3D可视化技术发现姿势问题、预防运动损伤。用户上传正面和侧面跑步视频后，系统自动识别人体关键骨骼点，生成3D人体模型还原跑步动作，逐帧分析着地瞬间、支撑中期、蹬伸阶段的关节角度，评估膝盖内扣、骨盆摇摆、身体前倾、步频步幅协调性等关键指标，最终输出可视化分析报告和针对性改进建议。
- 目标用户：业余跑者、跑步爱好者、健身教练
- 核心价值：让隐性的跑步姿势问题变得直观可见，用3D交互降低运动损伤风险

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册 | 上传视频、查看3D分析、生成报告 |

### 2.2 功能模块
1. **首页（上传与分析页）**：视频上传入口、3D模型实时渲染、帧播放控制、关键指标仪表盘
2. **分析报告页**：可视化问题摘要、关节角度图表、改进建议和训练动作推荐

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | 视频上传区 | 支持上传正面和侧面跑步视频，展示上传进度和预览缩略图 |
| 首页 | 3D模型渲染区 | Three.js渲染3D人体骨骼模型，支持360度旋转、缩放，用颜色标注问题部位（红色=严重、橙色=注意、绿色=正常） |
| 首页 | 帧播放控制条 | 逐帧播放、慢动作（0.25x/0.5x/1x）、暂停，标记关键帧（着地瞬间、支撑中期、蹬伸阶段） |
| 首页 | 关键指标仪表盘 | 实时显示膝盖内扣角度、骨盆摇摆幅度、身体前倾角度、步频步幅数据，用颜色和箭头标注状态 |
| 首页 | 阶段分析面板 | 三个跑步阶段的切换查看，每个阶段显示对应关节角度数据 |
| 报告页 | 问题摘要卡片 | 列出所有检测到的问题，按严重程度排序，含部位图示 |
| 报告页 | 关节角度折线图 | 全程关节角度变化曲线，标注异常区间 |
| 报告页 | 改进建议区 | 针对每个问题给出具体改进建议和2-3个推荐训练动作（含动作示意描述） |

## 3. 核心流程

用户进入首页 → 上传正面/侧面视频（或使用示例数据） → 系统模拟骨骼点识别 → 生成3D人体模型并播放跑步动画 → 用户拖拽旋转3D模型查看 → 使用帧控制逐帧分析 → 查看关键指标仪表盘 → 点击"生成报告" → 跳转报告页查看完整分析 → 获取改进建议和训练动作

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：深蓝黑底色（#0a0e1a），搭配电光蓝（#00d4ff）和活力橙（#ff6b35）作为强调色
- 辅助色：骨骼正常绿色（#00ff88）、警告橙色（#ffaa00）、危险红色（#ff3366）
- 按钮风格：圆角微发光按钮，hover时亮度提升和微缩放
- 字体：Oswald作为标题字体（运动感），Noto Sans SC作为正文字体
- 布局风格：左侧3D模型大区域 + 右侧指标面板，底部帧控制条
- 图标风格：线性运动主题图标（lucide-react）

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 首页 | 视频上传区 | 拖拽上传卡片，虚线边框，上传后显示缩略图预览 |
| 首页 | 3D模型渲染区 | 占据左侧2/3区域，深色背景网格地面，发光骨骼线条，问题部位高亮色球体标记 |
| 首页 | 帧播放控制条 | 底部时间轴滑块，播放/暂停按钮，速度选择器，关键帧标记点 |
| 首页 | 指标仪表盘 | 右侧1/3区域，4个圆形仪表盘（膝盖/骨盆/前倾/步频），数值+状态色 |
| 首页 | 阶段分析面板 | 3个tab切换（着地/支撑/蹬伸），对应关节角度数据卡片 |
| 报告页 | 问题摘要卡片 | 左侧列表，每张卡片含图标+问题名+严重度标签+简述 |
| 报告页 | 关节角度图表 | 中央大面积折线图，鼠标悬停显示数值，异常区间红色背景 |
| 报告页 | 改进建议区 | 右侧卡片列表，每个建议含动作名称+步骤说明+难度标签 |

### 4.3 响应式
- 桌面优先设计，1920x1080为最佳体验
- 平板端：3D区域和指标面板上下布局
- 移动端：简化视图，隐藏3D旋转功能，保留核心数据

### 4.4 3D场景设计
- 环境：深色科技感场景，地面网格线（蓝色），微弱环境雾
- 灯光：主方向光（冷白色）+ 环境光（蓝调）+ 骨骼自发光
- 相机：默认45度侧视角，支持OrbitControls自由旋转，滚轮缩放
- 骨骼渲染：圆柱体连接关节点，球体标记关节，正常部位青绿色，问题部位红/橙色脉冲发光
- 动画：预设跑步周期关键帧插值动画，支持慢动作播放
- 后处理：可选bloom发光效果增强科技感
